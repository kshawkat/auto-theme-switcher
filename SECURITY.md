# Security Policy

## Supported Versions

This project is maintained by a single owner and released as versioned builds rather than parallel long-term-support branches. Only the latest published release receives security fixes.

| Version | Supported |
| --- | --- |
| 2.5.0.0 (latest) | :white_check_mark: |
| Older releases | :x: |

If you are on an older version, upgrade to the latest release before reporting a vulnerability so it can be confirmed against current code.

## Reporting a Vulnerability

Do not open public issues for suspected vulnerabilities.

Report privately through GitHub's private vulnerability reporting (Security tab → Report a vulnerability) — enabled for this repository — or email the maintainer directly. You will receive an acknowledgement, and any fix will be coordinated before public disclosure.

Include:

- Affected version or commit
- Steps to reproduce
- Impact and affected functionality
- Suggested remediation, if known
